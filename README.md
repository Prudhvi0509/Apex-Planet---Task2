ApexPlanet Web Development: Interactive Responsive Portal

📌 Project Overview
Developed as part of the ApexPlanet Software Pvt. Ltd. internship program, this project showcases a modern, responsive user interface built from scratch. It integrates core front-end technologies to solve real-world interaction patterns, such as dynamic task management and secure form handling.

🌐 Live Deployment
https://prudhvi0509.github.io/Apex-Planet---Task2/

🛠️ Technical Stack
Architecture: Semantic HTML5 for SEO-friendly and accessible page structure.

Styling: CSS3 utilizing CSS Grid and Flexbox for a fluid, mobile-first design.

Logic: Vanilla JavaScript (ES6) for asynchronous-style DOM updates and client-side validation.

Version Control: Managed via Git and hosted on GitHub Pages.

🚀 Key Functionalities
1. Dynamic To-Do Application
Live DOM Manipulation: Users can add tasks which are immediately rendered to the UI without page refreshes.

State Management: Features a deletion toggle for each list item to manage active tasks effectively.

2. Intelligent Contact System
Validation Engine: Implements Regex-based email verification and presence checks to ensure data integrity.

User Feedback: Integrated alert systems provide immediate confirmation upon successful form submission.

3. Responsive UI/UX
Adaptive Navigation: A custom-coded navbar that transitions seamlessly from desktop to mobile views.

Visual Design: Professional aesthetic featuring linear gradients (#667eea to #764ba2) and high-readability typography.
